# Events Manager Plugin

A comprehensive WordPress plugin for managing events with custom post types, taxonomies, RSVP functionality, and REST API integration.

# Features

- Custom Post Type "Event" with full capabilities
- Custom Taxonomy "Event Type"
- Custom meta boxes for event details (date, location, capacity)
- RSVP functionality for logged-in users
- Advanced admin list view with custom columns
- Front-end templates for single and archive views
- Shortcode `[events_list]` for embedding event listings
- Search and filtering system
- Email notifications for new events
- REST API endpoints for external applications
- Fully internationalized
- Security best practices implemented
- Performance optimized
- Unit tests included


# Installation Using WP-CLI

```bash
# Navigate to plugins directory
cd wp-content/plugins/

# Clone or copy the plugin
git clone https://github.com/O-thedev/events-manager

# Activate the plugin
wp plugin activate events-manager
```
